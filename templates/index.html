API endpoints

Get-quote

Get a random quote in a JSON format. The quotes are generated by the Ollama container and stored in a file. If the file is empty, a fallback file will be used.

{ "quote": "Some quote text", "author": "Some Name" }

Full URL: http://localhost:5000/api/get-quote


GET QUOTE

Get-image

Get a random 1024x1024px jpeg image. The images are fetched from Unsplash, if the site is not available, some fallback images will be used.

Full URL: http://localhost:5000/api/get-image


GET IMAGE

Get-final

Get the final sharepic as a png file. It combines the quote and the image. Optionally the following parameters can be used:

dark: The font is white and the image is slightly dimmed

fancy: The font is switched to be more fancyfull

Full URL: http://localhost:5000/api/get-final


GET FINAL

Fetch-quotes

Fetch a quote from the Ollama backend and store it in a file. Because the process can be quite slow, this is currently disabled from production.

Full URL: http://localhost:5000/api/fetch-quotes


GET QUOTE
